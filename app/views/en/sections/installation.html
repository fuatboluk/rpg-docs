Installation

RPG Framework does not require any additional setup tools or dozens of PHP libraries. Simply download the files from the GitHub repository and copy them to the default directory of your web server. You can do this in two different ways:

Using the git command

In the command line, navigate to the default directory of your web server:

cd /var/www/html
{{note}}

Adjust the command according to the default working directory of your web server for your platform!

We will clone the RPG Framework into the directory we are in using the git command:

git clone https://github.com/fuatboluk/rpg-framework

If everything went smoothly, a new directory named "rpg-framework" will be created within the directory. You can rename this directory according to your project's name if you wish.

By downloading an archive file

Now, go to rpg-framework and download the latest RPG version as an archive. Then, extract the archive file into the default directory of your web server.
