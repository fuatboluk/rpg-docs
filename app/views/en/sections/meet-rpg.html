Meet RPG

RPG is a small and fast framework. Its syntax is very simple and it comes with minimal features, which makes it easy to learn in a short time. Using RPG is like building a puzzle. You can break down even large projects into as many small pieces as you wish.

Build your project in an organized, fast, and manageable way by using the MVC software design pattern. The RPG Framework has a structure that completely abstracts PHP and HTML resources from each other. It is possible to write inline PHP code and create loops in the view files. However, the syntax of RPG encourages all background operations, including loops, to be handled in the controller. This way, the front-end and back-end are completely abstracted from each other.

If you're new to using a PHP framework or familiar with popular frameworks, you can quickly and easily learn RPG Framework. We believe that, whether alone or with your team, you will accomplish great things using RPG Framework.
