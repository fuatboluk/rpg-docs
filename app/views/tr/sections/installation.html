Kurulum

RPG Framework'ün yardımcı kurulum araçlarına ve onlarca PHP kütüphanesine ihtiyacı yoktur. Sadece GitHub deposundan dosyaları indirin ve kullandığınız web sunucusunun varsayılan dizinine kopyalayın. Bunu iki farklı yöntemle yapabilirsiniz:

Git Komutunu Kullanarak

Komut satırında, web sunucunuzun varsayılan dizinine gidin:

cd /var/www/html
{{note}}

Kullandığınız web sunucusunun, platformunuz için ön tanımlı çalışma dizininine göre komutu düzenleyin!

Git komutunu kullanarak RPG Framework'ü bulunduğumuz dizine klonlayacağız:

git clone https://github.com/fuatboluk/rpg-framework

Her şey yolunda gittiyse, dizin içinde "rpg-framework" adında yeni bir dizin oluşacaktır. Dilerseniz bu dizini projenizin ismine göre yeniden adlandırabilirsiniz.

Arşiv dosyası indirerek

Şimdi, rpg-framework adresine giderek en son RPG sürümünü arşiv olarak indirin. Ardından, arşiv dosyasını web sunucunuzun varsayılan dizinine çıkarın.
